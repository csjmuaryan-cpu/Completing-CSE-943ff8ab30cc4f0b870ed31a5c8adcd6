Experiment 2

Objective: Write a Java program that accepts two integer numbers as command-line arguments, calculates their sum, and displays the result.

Program:

public class CommandLineSum {
    public static void main(String[] args) {
        if (args.length < 2) {
            System.out.println("Please provide two integers.");
            return;
        }
        try {  
            int num1 = Integer.parseInt(args[0]);  
            int num2 = Integer.parseInt(args[1]);
            System.out.println("Sum is: " + (num1 + num2));  
        } catch (NumberFormatException e) {
            System.out.println("Error: Invalid integers.");
        }
    }
}

Scenario 1 (Valid): java CommandLineSum 10 25 -> Sum is: 35

Scenario 2 (Invalid): java CommandLineSum 5 five -> Error: Invalid integers.

Experiment 3

Objective: Write a Java program to create two classes: Student and Rectangle, with methods to display their details and area respectively.

Program:

class Student { 
    String name; int rollNo;
    Student(String n, int r) { name = n; rollNo = r; }

        System.out.println("Name: " + name + ", Roll: " + rollNo);
    }  
}
class Rectangle { 
    double length, width;
    Rectangle(double l, double w) { length = l; width = w; }
    void displayArea() {
        System.out.println("Area: " + (length * width));
    }
}

Output:

Name: Alice, Roll: 101
Area: 15.0

Experiment 5

Objective: Write a Java program that demonstrates exception handling (try-catch-finally) and multithreading.

Program:

public class ExceptionAndThreadDemo {
    public static void main(String[] args) {
        try { int r = 10 / 0; }
        catch (ArithmeticException e) { System.out.println("Error: / by zero"); }
        finally { System.out.println("Finally block executed."); }
        new Thread(() -> System.out.println("Thread 1 running")).start();
        new Thread(() -> System.out.println("Thread 2 running")).start();
    }
}

Output (threading order may vary):

Error: / by zero
Finally block executed.
Thread 1 running
Thread 2 running

Experiment 7

Objective: Use Java I/O streams to read from `input.txt` and write its content to `output.txt`.

Program:

import java.io.*;
public class FileIOExample {
    public static void main(String[] args) {
        try (Reader r = new FileReader("input.txt");
             Writer w = new FileWriter("output.txt")) {
            int c;
            while ((c = r.read()) != -1) { w.write(c); }
        } catch (IOException e) { e.printStackTrace(); }
    }
}

Result: After running, `output.txt` will contain a copy of the content from `input.txt`.

Experiment 9
