Objective: Write a simple Java program using Eclipse IDE to display the message "Hello, Java World from Eclipse!" on the console, also use all major escape sequences including \n (newline), \t (tab), \\ (backslash), \" (double quote), \' (single quote), and \r (carriage return).

Program:

public class EscapeDemo {
    public static void main(String[] args) {
        // Basic message
        System.out.println("Hello, Java World from Eclipse!");
        
        // Using newline (\\n)
        System.out.println("\nThis is printed on a new line.");
        
        // Using tab (\\t) 
        System.out.println("Name:\tJohn Doe"); 
        System.out.println("Roll:\t118");
        
        // Using backslash (\\\\)
        System.out.println("File path: C:\\Program Files\\Java\\jdk");
        
        // Using double quote (\\")
        System.out.println("She said, \"Java is awesome!\"");
        
        // Using single quote (\\')
        System.out.println("It's a good day to learn Java.");
        
        // Using carriage return (\\r) 
        System.out.println("Learning Java is fun!\rStart: ");
    }  
}

Output:

This is printed on a new line.

Roll:   118

Start: Java is fun!

Objective: Write a Java program that accepts two integer numbers as command-line arguments, calculates their sum, and displays the result.

Program:

public class CommandLineSum {
    public static void main(String[] args) {
        if (args.length < 2) {
            System.out.println("Please provide two integer numbers.");
            return;
        }
        try {  
            int num1 = Integer.parseInt(args[0]);  
            int num2 = Integer.parseInt(args[1]);
            int sum = num1 + num2;
            System.out.println("Sum of " + num1 + " and " + num2 + " is: " + sum);  
        } catch (NumberFormatException e) {
            System.out.println("Error: Please enter valid integer numbers.");
        }
    }
}

Scenario 1: Command: java CommandLineSum 10 25

Output: Sum of 10 and 25 is: 35

Scenario 2: Command: java CommandLineSum 10

Output: Please provide two integer numbers.

Scenario 3: Command: java CommandLineSum 5 five

Output: Error: Please enter valid integer numbers.

        Dog dog = new Dog(); 
        dog.bark(); 
        dog.bark("loud"); 
        dog.bark(3);
    }
}

Objective: Write a Java program to demonstrate exception handling and multithreading.

Program:

public class ExceptionAndThreadDemo {
    public static void main(String[] args) {
        System.out.println("--- Exception Handling Demo ---");
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("Error: Cannot divide by zero.");
        } finally {
            System.out.println("The 'finally' block is always executed.");
        }
        
        System.out.println("\n--- Multithreading Demo ---");
        Thread thread1 = new Thread(new MessagePrinter("Thread-1"));
        Thread thread2 = new Thread(new MessagePrinter("Thread-2"));
        thread1.start();
        thread2.start();
    }
}
class MessagePrinter implements Runnable {
    private String threadName;
    public MessagePrinter(String name) { this.threadName = name; }
    public void run() {
        for (int i = 1; i <= 3; i++) {
            System.out.println(threadName + " is running, message #" + i);
            try { Thread.sleep(10); } catch (InterruptedException e) {}
        }
    }
}

Output:

--- Exception Handling Demo ---
Error: Cannot divide by zero.
The 'finally' block is always executed.

--- Multithreading Demo ---
Thread-1 is running, message #1
Thread-2 is running, message #1
Thread-1 is running, message #2
Thread-2 is running, message #2
Thread-1 is running, message #3
Thread-2 is running, message #3

Objective: Write a Java program that uses Java I/O streams to read from and write to text files.

Program:

import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;

public class FileIOExample {
    public static void main(String[] args) {
        try (FileWriter writer = new FileWriter("input.txt")) {
            writer.write("Hello from input.txt!");
        } catch (IOException e) { e.printStackTrace(); }

        try (FileReader reader = new FileReader("input.txt");
             FileWriter writer = new FileWriter("output.txt")) {
            int character;
            while ((character = reader.read()) != -1) {
                writer.write(character);
            }
        } catch (IOException e) { e.printStackTrace(); }
        System.out.println("File copy complete.");
    }
}

Output:

  • A file input.txt is created with content.
  • A file output.txt is created with a copy of input.txt's content.
  • Console prints: File copy complete.

public class MessageService {
    public String getMessage() { return "Message from injected service!"; }
}

File 3: App.java

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.context.ApplicationContext;
import org.springframework.context.annotation.*;
@Configuration @ComponentScan(".")
public class App {
    @Autowired private MessageService service;
    public void displayMessage() { System.out.println(service.getMessage()); }
    public static void main(String[] args) {
        ApplicationContext ctx = new AnnotationConfigApplicationContext(App.class);
        ctx.getBean(App.class).displayMessage();
    }
}

Output:

Message from injected service!
