### **Experiment: 5**

**Objective:** Write a Java program that demonstrates exception handling and multithreading.

**Program:**
```java
public class ExceptionAndThreadDemo {
    public static void main(String[] args) {
        System.out.println("--- Exception Handling Demo ---");
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("Error: Cannot divide by zero.");
        } finally {
            System.out.println("The 'finally' block is always executed.");
        }
        System.out.println("\n--- Multithreading Demo ---");
        Thread thread1 = new Thread(new MessagePrinter("Thread-1"));
        Thread thread2 = new Thread(new MessagePrinter("Thread-2"));
        thread1.start();
        thread2.start();
    }
}
class MessagePrinter implements Runnable {
    private String threadName;
    public MessagePrinter(String name) { this.threadName = name; }
    @Override
    public void run() {
        for (int i = 1; i <= 3; i++) {
            System.out.println(threadName + " is running, message #" + i);
            try { Thread.sleep(10); } catch (InterruptedException e) { }
        }
    }
}
```

**Output:**
```
--- Exception Handling Demo ---
Error: Cannot divide by zero.
The 'finally' block is always executed.

--- Multithreading Demo ---
Thread-1 is running, message #1
Thread-2 is running, message #1
Thread-1 is running, message #2
Thread-2 is running, message #2
Thread-1 is running, message #3
Thread-2 is running, message #3
```

**Experiment: 7**

**Objective:** Write a Java program that uses Java I/O streams to read from and write to text files.

**Program:**
```java
import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;

public class FileIOExample {
    public static void main(String[] args) {
        try (FileWriter writer = new FileWriter("input.txt")) {
            writer.write("Hello from input.txt!");
        } catch (IOException e) { e.printStackTrace(); }

        try (FileReader reader = new FileReader("input.txt");
             FileWriter writer = new FileWriter("output.txt")) {
            int character;
            while ((character = reader.read()) != -1) {
                writer.write(character);
            }
        } catch (IOException e) { e.printStackTrace(); }
        System.out.println("File copy complete.");
    }
}
```
**Output:**
*   A file `input.txt` is created with content.
*   A file `output.txt` is created with a copy of `input.txt`'s content.
*   Console prints: `File copy complete.`
